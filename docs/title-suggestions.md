# タイトル案

## 現在のタイトル
- ヘッダー: 「パフォーマンスDB | Performance Records」
- ホームページ: 「Performance Training Database」

## 提案案

### 1. シンプル・短縮版
- **「パフォDB」** / **「PerfDB」**
  - 短くて覚えやすい
  - カジュアルで親しみやすい

### 2. 機能重視
- **「トレーニングDB」** / **「TrainingDB」**
  - 機能が明確
  - シンプルで分かりやすい

### 3. ブランド名風
- **「PerfTrack」** / **「TrainTrack」**
  - トレーニングを追跡するイメージ
  - 英語圏でも通じやすい

### 4. 日本語のみ（シンプル）
- **「パフォDB」**
  - 日本語ユーザー向け
  - 短くて覚えやすい

### 5. 英語のみ（シンプル）
- **「PerfDB」**
  - グローバル対応
  - 短くて覚えやすい

### 6. ハイブリッド（短縮版）
- **「パフォDB | PerfDB」**
  - 日英両対応
  - 現在の形式を短縮

## 推奨案

**「パフォDB」** または **「PerfDB」**

理由：
- 短くて覚えやすい
- 入力しやすい
- ブランドとしても機能する
- モバイルでも表示しやすい
