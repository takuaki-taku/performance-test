# デザイン・機能議論ドキュメント

## 📋 プロジェクト概要

**PerfDB** - ジュニアテニス選手向けのトレーニング管理アプリケーション

## 🎯 重要なポイント（優先順位順）

### 1. 簡単にアクセスできる ⭐⭐⭐
**目標**: 選手が迷わず、素早く目的のトレーニングにアクセスできる

**現状**:
- ✅ ホームページに4つのトレーニングカテゴリをカード形式で表示
- ✅ クイックアクセスセクション（続きから始める、マイページ）
- ✅ ヘッダーに主要ナビゲーション

**改善案・議論点**:
- [ ] よく使うトレーニングを「お気に入り」として保存できる機能
- [ ] 検索機能の追加（トレーニング名で検索）
- [ ] 最近実施したトレーニングへのクイックアクセス
- [ ] モバイルでの操作性向上（スワイプ操作、タッチ最適化）
- [ ] オフライン対応（キャッシュ機能）

### 2. わかりやすい、インストラクション ⭐⭐⭐
**目標**: 選手が一人でも正しくトレーニングを実施できる

**現状**:
- ✅ 各トレーニングに画像、説明、実施方法を表示
- ✅ タイマー機能（ストレッチ時間の計測）
- ✅ チェックポイントの表示

**改善案・議論点**:
- [ ] 動画インストラクションの追加
- [ ] ステップバイステップガイド（段階的な説明）
- [ ] よくある間違いとその修正方法
- [ ] 音声ガイダンス（「もう少し深く」「もう少し長く」など）
- [ ] イラストやアニメーションでの視覚的な説明
- [ ] 多言語対応（英語、日本語）
- [ ] 年齢別・レベル別のインストラクション

### 3. トレーニングの評価を記録できる ⭐⭐⭐
**目標**: 選手の進捗を可視化し、成長を実感できる

**現状**:
- ✅ 評価レベル（要改善/達成/優秀/開始済み）の記録
- ✅ トレーニング実施日の記録
- ✅ マイページで進捗サマリーを表示
- ✅ コーチが管理画面で評価を付ける機能

**改善案・議論点**:
- [ ] 選手自身も評価を付けられる機能（自己評価）
- [ ] 評価の履歴をグラフで表示（時系列での成長）
- [ ] 目標設定機能（「今月は10個のトレーニングを達成レベルにする」）
- [ ] バッジ・アチーブメントシステム（モチベーション向上）
- [ ] 評価に対するコメント機能（詳細なフィードバック）
- [ ] 写真や動画の添付（フォーム確認用）

### 4. 自分のトレーニングを投稿できる場所 ⭐⭐
**目標**: 選手が自主的にトレーニングを記録・共有できる

**現状**:
- ✅ トレーニング詳細ページで「今日このトレーニングを行ったことを記録する」機能
- ✅ メモ（コメント）の記録
- ⚠️ コーチからのフィードバック機能は未実装

**改善案・議論点**:
- [ ] 専用の「投稿」ページまたはセクション
- [ ] 写真・動画のアップロード機能
- [ ] トレーニング日記機能（日々の記録を時系列で表示）
- [ ] トレーニングのカスタマイズ（自分でアレンジした方法を記録）
- [ ] トレーニングの共有機能（他の選手と共有、ただしプライバシー考慮）
- [ ] 投稿の下書き保存機能

### 5. コーチからのフィードバックもらえる ⭐⭐
**目標**: コーチが選手のトレーニングを確認し、適切なアドバイスを提供できる

**現状**:
- ✅ コーチ（管理者）が管理画面で評価を付ける機能
- ⚠️ コメント機能はデータベースに存在するが、UIで表示・編集が不完全
- ❌ 選手側でコーチのフィードバックを確認する機能が未実装

**改善案・議論点**:
- [ ] 選手のマイページに「コーチからのフィードバック」セクションを追加
- [ ] フィードバック通知機能（新しいフィードバックが来たら通知）
- [ ] フィードバックへの返信機能（選手が質問できる）
- [ ] フィードバックの履歴表示
- [ ] コーチが選手の投稿に直接コメントできる機能
- [ ] フィードバックの優先度設定（重要/通常/参考）
- [ ] 音声メッセージでのフィードバック

## 🎨 デザインに関する議論

### UI/UXデザイン

#### カラースキーム
**現状**: 
- ウォームアップ: オレンジ/赤
- 体幹: 青/インディゴ
- 柔軟性: 緑/エメラルド
- クールダウン: 紫/青

**議論点**:
- [ ] ジュニア向けにもっと明るい・楽しい色合いにするか
- [ ] 色の意味を統一（例：緑=達成、赤=要改善）
- [ ] ダークモード対応

#### レイアウト
**現状**: 
- カードベースのレイアウト
- レスポンシブデザイン対応

**議論点**:
- [ ] より視覚的に魅力的なデザイン（イラスト、アイコン）
- [ ] アニメーション効果の追加（楽しい雰囲気）
- [ ] タブレット最適化（大きな画面での表示）

#### タイポグラフィ
**議論点**:
- [ ] 読みやすいフォントサイズ（ジュニア向けに大きめ）
- [ ] ひらがな・カタカナの使用（漢字を減らす）
- [ ] 絵文字やアイコンの活用

## 📝 メモ・アイデア

### アイデア1: トレーニング日記
選手が日々のトレーニングを日記形式で記録。コーチがコメントを付ける。

週間・月間のチャレンジを設定し、達成するとバッジを獲得。
